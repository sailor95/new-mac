# new-mac

What to do after getting a new Mac

## Chores

- Pull default Apple apps into one folder in Launchpad
- Setup Finder
  - Toolbar appearance
- Setup Dock
  - Only leave min necessary apps
  - On left
  - Auto hide
  - Resize
- Setup find my Mac
